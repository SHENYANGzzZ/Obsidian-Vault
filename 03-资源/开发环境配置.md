# 开发环境配置指南

> 完整的开发环境搭建和配置指南

## 🖥️ 操作系统配置

### macOS 推荐配置
```bash
# 安装 Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装常用工具
brew install git node python3 wget curl

# 安装开发工具
brew install --cask visual-studio-code iterm2 postman docker
```

### 环境变量配置
```bash
# ~/.zshrc 或 ~/.bash_profile
export PATH="/usr/local/bin:$PATH"
export PATH="$HOME/.local/bin:$PATH"
export EDITOR='code'
```

## 📦 包管理器

### Node.js 生态
```bash
# 安装 nvm（Node版本管理）
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 安装 Node.js
nvm install node
nvm use node

# 全局包
npm install -g yarn pnpm typescript
```

### Python 生态
```bash
# 安装 pyenv（Python版本管理）
brew install pyenv

# 安装 Python
pyenv install 3.11.0
pyenv global 3.11.0

# 虚拟环境
python -m venv myenv
source myenv/bin/activate
```

## 🐳 容器化环境

### Docker 配置
```bash
# 安装 Docker Desktop
brew install --cask docker

# 常用命令
docker --version
docker-compose --version
```

### Docker Compose 示例
```yaml
version: '3.8'
services:
  app:
    build: .
    ports:
      - "3000:3000"
    volumes:
      - .:/app
    depends_on:
      - db
  db:
    image: postgres:14
    environment:
      POSTGRES_DB: mydb
      POSTGRES_USER: user
      POSTGRES_PASSWORD: password
```

## 🔐 安全配置

### SSH 配置
```bash
# 生成 SSH 密钥
ssh-keygen -t ed25519 -C "your_email@example.com"

# 添加到 SSH 代理
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519
```

### GPG 配置
```bash
# 安装 GPG
brew install gnupg

# 生成密钥
gpg --full-generate-key

# 配置 Git 使用 GPG
git config --global commit.gpgsign true
git config --global user.signingkey YOUR_KEY_ID
```

## 🚀 自动化配置

### Dotfiles 管理
```bash
# 使用 GNU Stow 管理配置文件
brew install stow

# 目录结构
dotfiles/
├── git/
│   └── .gitconfig
├── zsh/
│   └── .zshrc
└── vscode/
    └── settings.json
```

### 安装脚本
```bash
#!/bin/bash
# setup.sh - 一键配置开发环境

echo "🚀 开始配置开发环境..."

# 安装 Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装工具
brew install git node python3
brew install --cask visual-studio-code iterm2

# 配置 Git
git config --global user.name "Your Name"
git config --global user.email "your_email@example.com"

echo "✅ 开发环境配置完成！"
```
