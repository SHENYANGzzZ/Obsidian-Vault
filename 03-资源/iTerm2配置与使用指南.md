# iTerm2配置与使用指南

> 终端利器，提升开发效率的快捷键和配置技巧

## 📑 标签页与分屏管理

### 标签页操作
| 快捷键 | 功能 |
|--------|------|
| `⌘ + T` | 新建标签页 |
| `⌘ + W` | 关闭当前标签/分屏 |
| `⌘ + ]` | 切换到下一个标签页 |
| `⌘ + [` | 切换到上一个标签页 |
| `⌘ + 数字` | 跳转到指定标签页（1-9） |

### 分屏操作
| 快捷键 | 功能 |
|--------|------|
| `⌘ + D` | 垂直分屏 |
| `⌘ + Shift + D` | 水平分屏 |
| `⌘ + 方向键` | 在分屏间切换 |
| `⌘ + Option + 方向键` | 调整分屏大小 |

### 使用场景
- **垂直分屏**：同时查看代码和日志
- **水平分屏**：对比两个文件或监控多个服务
- **标签页**：不同项目或环境的隔离

## 🖱️ 光标与文本选择

### 光标移动
| 快捷键 | 功能 |
|--------|------|
| `⌘ + ←` | 跳到行首 |
| `⌘ + →` | 跳到行尾 |
| `⌥ + ←` | 按单词向左移动 |
| `⌥ + →` | 按单词向右移动 |
| `Ctrl + A` | 跳到行首（Emacs风格） |
| `Ctrl + E` | 跳到行尾（Emacs风格） |

### 文本选择
| 快捷键 | 功能 |
|--------|------|
| `⌘ + A` | 全选 |
| `Shift + 方向键` | 扩展选择 |
| `⌥ + Shift + 方向键` | 按单词扩展选择 |

### 实用技巧
- **智能选择**：三击选中整行，四击选中整段
- **矩形选择**：`⌥ + 拖动` 进行列选择

## 📋 剪贴板增强

### 复制粘贴
| 快捷键 | 功能 |
|--------|------|
| `⌘ + C` | 复制选中文本 |
| `⌘ + V` | 粘贴 |
| `⌘ + Shift + C` | 复制（无格式） |
| `⌘ + Option + V` | 打开粘贴历史 |

### 剪贴板历史
- **查看历史**：`⌘ + Shift + H`
- **快速粘贴**：从历史中选择并粘贴
- **清除历史**：在偏好设置中管理

### 使用场景
- 复制多个命令依次执行
- 在不同终端窗口间共享内容
- 保存常用命令片段

## 🔍 实用功能

### 清屏与搜索
| 快捷键 | 功能 |
|--------|------|
| `⌘ + R` | 清屏并滚动到提示符 |
| `⌘ + K` | 清空整个缓冲区 |
| `⌘ + F` | 搜索（支持正则） |
| `⌘ + G` | 查找下一个 |
| `⌘ + Shift + G` | 查找上一个 |

### 搜索技巧
- **正则搜索**：在搜索框中勾选"Regex"
- **高亮显示**：搜索结果会高亮标记
- **快速跳转**：使用 `⌘ + G` 在匹配间切换

### 全屏与显示
| 快捷键 | 功能 |
|--------|------|
| `⌘ + Enter` | 全屏/退出全屏 |
| `⌘ + +` | 增大字体 |
| `⌘ + -` | 减小字体 |
| `⌘ + 0` | 重置字体大小 |
| `⌘ + ,` | 打开偏好设置 |

### 自动补全
| 快捷键 | 功能 |
|--------|------|
| `⌘ + ;` | 自动补全历史命令 |
| `Tab` | 命令/文件名补全 |
| `Ctrl + R` | 反向搜索历史命令 |

## ⚡ 高级特性

### 热键窗口（Hotkey Window）
- **设置**：`Preferences → Keys → Hotkey`
- **功能**：全局热键快速呼出/隐藏终端
- **推荐**：设置 `⌥ + Space` 或 `⌃ + ~`
- **优势**：无需切换应用，随时访问终端

### Profiles（配置文件）
- **用途**：为不同服务器/环境创建不同配置
- **设置**：`Preferences → Profiles`
- **示例**：
  - 生产服务器：红色背景（警示）
  - 开发环境：绿色背景
  - 测试环境：黄色背景

### Triggers（触发器）
- **功能**：根据输出自动执行动作
- **设置**：`Preferences → Profiles → Advanced → Triggers`
- **示例**：
  - 错误高亮：匹配 `Error|Exception` 并高亮
  - 自动弹窗：匹配 `password:` 时弹出通知
  - 声音提醒：匹配特定模式时播放声音

### Shell Integration
- **启用**：`Preferences → Profiles → General → Command`
- **功能**：
  - 命令完成标记（绿色箭头）
  - 命令持续时间显示
  - 当前工作目录提示
  - 命令历史增强

### 时间回滚（Instant Replay）
| 快捷键 | 功能 |
|--------|------|
| `⌘ + Option + B` | 开启时间回滚 |
| `⌘ + Option + 左右方向键` | 回放/快进 |
| `Esc` | 退出回滚模式 |

**用途**：查看之前输出的内容，即使已经滚动过去

## ⚙️ 推荐配置

### 必开选项
1. **智能光标颜色**：`Preferences → General → Magic → Enable smart cursor color`
2. **智能选择**：`Preferences → General → Selection → Enable smart selection`
3. **无限滚动**：`Preferences → Profiles → Terminal → Scrollback Buffer → Unlimited`
4. **Shell Integration**：启用后体验大幅提升

### 字体推荐
- **等宽字体**：JetBrains Mono, Fira Code, Hack, Source Code Pro
- **大小**：14-16px（根据屏幕调整）
- **连字支持**：启用 `Ligatures` 选项

### 配色方案
- **内置推荐**：Solarized Dark, Dracula, Nord
- **自定义**：`Preferences → Profiles → Colors → Color Presets`
- **在线资源**：[iTerm2 Color Schemes](https://iterm2colorschemes.com/)
