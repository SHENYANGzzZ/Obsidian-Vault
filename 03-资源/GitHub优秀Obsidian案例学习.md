# GitHub 优秀 Obsidian 案例学习

  - Obsidian Tasks 使用
  - Meta Bind 集成
  - 社区创建的启动模板

### 2. thor314/obsidian-setup

  - 约 100 个插件的使用建议

### 3. DuskWasHere/dusk-obsidian-vault

### 4. SoRobby/ObsidianStarterVault

## 🎯 2026 年最佳实践

### 同步方案
- ✅ 使用 GitHub 进行免费同步（替代付费 Obsidian Sync）
- ✅ 创建私有仓库
- ✅ 使用 Git 插件配置

### 插件策略
- ⚠️ 避免过度依赖插件
- ✅ 专注于必要插件：
  - **Remotely Save**: 跨设备同步
  - **Mind Map**: 可视化
  - **Full Calendar**: 日历视图
  - **Day Planner**: 日程规划
  - **Easy Typing**: 智能格式化

### 组织结构
- ✅ 从简单开始，让结构自然演化
- ✅ 使用 PARA 或 Zettelkasten 框架
- ❌ 避免一开始就建立过于复杂的系统

### 模板使用
- ✅ 从一开始就使用模板
- ✅ 确保一致性
- ✅ 便于 Dataview 等插件集成

### 核心原则
- 🎯 保持简单
- 🎯 让系统"自然运作"
- 🎯 专注于思考，而非系统本身
