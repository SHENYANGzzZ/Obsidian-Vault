## 第三步：掌握基本操作

### 3.1 捕获想法

当你有一个想法时：

1. 打开 `00-收集箱`
2. 创建新笔记
3. 使用 [[闪念笔记模板]]
4. 写下核心想法
5. 标记 `#闪念笔记`

### 3.2 处理 Inbox

每天至少一次：

1. 打开 `00-收集箱`
2. 查看所有闪念笔记
3. 问自己：这条有价值吗？链接到哪里？
4. 转化为永久笔记或删除

详见：[[收集箱处理工作流]]

### 3.3 建立链接

Obsidian 的核心是双向链接：

- 输入 `[[` 触发链接建议
- 输入 `#` 触发标签建议
- 点击链接跳转到目标笔记
- 使用 `Ctrl/Cmd + O` 快速打开笔记

详见：[[笔记链接的艺术]]

## 第四步：日常工作流程

### 每日流程

```dataview
TABLE 
WHERE file.cday = date(today)
```

1. **晨间（10分钟）**
   - 处理 Inbox
   - 计划今日重点

2. **日间**
   - 捕获任何想法到 Inbox
   - 阅读时做文献笔记

3. **晚间（15分钟）**
   - 写每日笔记
   - 复盘今日收获
   - 更新明日计划

详见：[[每日笔记模板]]

### 每周流程

1. 回顾本周 Inbox
2. 更新相关 MOC
3. 整理归档
4. 计划下周

## 第五步：进阶技巧

### 使用 MOC

MOC（内容地图）帮助你组织主题：

1. 找到相关主题的 MOC
2. 在 MOC 中添加你的笔记链接
3. 通过 MOC 导航整个主题

### 使用标签

标签用于分类和筛选：

- `#永久笔记` - 经过思考的笔记
- `#闪念笔记` - 临时想法
- `#项目/项目名` - 项目标签

详见：[[标签管理规范]]

### 使用模板

使用模板保持一致性：

- 按 `Ctrl/Cmd + P` 打开命令面板
- 输入 `模板` 选择模板
- 选择合适的模板创建笔记
