## 工具与配置

### OpenCode 会话记录路径

```
~/.claude/history.jsonl
```

存储格式：`{"display": "...", "timestamp": ..., "project": "...", "sessionId": "..."}`

- 只记录用户输入，不记录 AI 回答
- 可通过 `project` 字段按项目过滤
- 结合 git log 可还原完整的开发上下文

### Obsidian 仓库结构

