# 坐标转换

**项目**：[[water-web]]
**标签**：#技术笔记 #坐标系 #water-web

---

## 概念

### 坐标系对比

| 坐标系       | 代号      | 格式                | 中国范围                   |
| ------------ | --------- | ------------------- | -------------------------- |
| WGS84 经纬度 | EPSG:4326 | (119.5, 34.7)       | 经度 73°~135°，纬度 3°~54° |
| Web 墨卡托   | EPSG:3857 | (13293816, 4125781) | ±20037508 米               |

> ⚠️ **关键发现**：墨卡托坐标数值范围与经纬度完全不同，不能混用判断！

---

## 转换公式

### 墨卡托 → 经纬度

```javascript
const MAX_EXTENT = 20037508.34; // Web 墨卡托最大范围

function mercatorToLatLng(x, y) {
  // 经度：线性映射
  const lng = (x / MAX_EXTENT) * 180;
  // 纬度：非线性映射（反三角函数）
  const lat =
    (2 * Math.atan(Math.exp((y / MAX_EXTENT) * Math.PI)) - Math.PI / 2) *
    (180 / Math.PI);
  return { lng, lat };
}
```

### 批量转换二维数组

```javascript
// 输入：[[x, y], [x, y], ...]
// 输出：[{ lng, lat }, { lng, lat }, ...]
const latLngArray = coordinates.map(([x, y]) => mercatorToLatLng(x, y));
```
