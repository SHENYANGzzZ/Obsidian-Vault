## 🔄 常用工作流

1. **每日工作流**

   - 晨间：查看 Inbox → 处理闪念笔记
   - 日间：捕获想法 → 记录到 Inbox
   - 晚间：写每日笔记 → 复盘

2. **知识管理流程**

   - 收集 → 处理 → 沉淀 → 连接

3. **项目工作流**
   - 创建项目 → 记录开发日志 → 完成任务 → 归档
