# 如何写好笔记标题

## 核心观点

好的标题是笔记被发现和理解的关键。一个优秀的标题应该能够独立传达笔记的核心内容。

## 标题的重要性

### 为什么标题很重要

1. **可发现性** - 在搜索和链接时快速定位
2. **独立性** - 不依赖上下文就能理解
3. **可链接性** - 让其他人愿意链接到这篇笔记
4. **可记忆性** - 方便日后回忆和引用

### 糟糕标题的问题

- 过于宽泛：如"关于思考的一些想法"
- 过于抽象：如"论方法论"
- 依赖上下文：如"续：上次的讨论"
- 过于冗长：把所有内容都塞进标题

## 好标题的特征

### 1. 具体且明确

```
❌ 差：写作技巧
✅ 好：如何用卡片笔记法提高写作效率
```

### 2. 陈述核心观点

```
❌ 差：关于链接的思考
✅ 好：链接比文件夹更重要
```

### 3. 使用主动语态

```
❌ 差：笔记的原子化
✅ 好：原子笔记的价值
```

### 4. 适度长度

- 中文标题：8-20 字为宜
- 英文标题：3-8 个单词
- 避免过短（信息不足）或过长（难以记忆）

## 实践技巧

### 从问题出发

好的标题往往回答一个问题：
- "如何..." → 方法类笔记
- "为什么..." → 原理类笔记
- "什么是..." → 概念类笔记

### 使用冒号结构

```
主题：具体观点
例：Zettelkasten：让知识产生复利的方法
```

### 让标题"可链接"

想象别人在写笔记时会这样引用：
- "正如 [[如何写好笔记标题]] 中所说..."

## 不同类型笔记的标题建议

| 笔记类型 | 标题风格 | 示例 |
|---------|---------|------|
| 概念笔记 | 名词短语 | 原子笔记的价值 |
| 方法笔记 | 如何... | 如何建立笔记链接网络 |
| 观点笔记 | 陈述句 | 链接比文件夹更重要 |
| 资源笔记 | 资源名 + 主题 | 《卡片笔记写作法》核心观点 |

## 常见错误

1. **使用日期作为标题** - 除非是日记
2. **使用编号** - 如"笔记001"
3. **复制文件名思维** - 笔记不是文件，是想法
4. **过度追求完美** - 可以先写后改

## 标题优化流程

1. 写完笔记后，问自己：这篇笔记的核心是什么？
2. 用一句话概括
3. 检查是否可以独立理解
4. 模拟别人链接时的语境
5. 必要时调整
