# Obsidian 知识库

## 项目简介

这是一个基于 Obsidian 的个人知识库，用来管理项目资料、每日记录、长期参考资料和阶段性归档。

## 当前目录结构

```text
Obsidian-Vault/
├── 00-收集箱/      # 临时收集，待后续处理
├── 01-项目/        # 项目主目录
├── 02-领域/        # 长期关注领域
├── 03-资源/        # 参考资料与方法文档
├── 04-归档/        # 已归档项目或历史资料
├── 05-每日/        # 每日原始记录入口
├── 06-模板/        # 模板与记录规范
├── MOCs/           # 索引与知识地图
└── .obsidian/      # Obsidian 配置
```

## 当前记录约定

- 每日执行内容先写入 `05-每日/`
- 项目相关内容按项目块记录，再按周整理到对应项目目录
- `01-项目/项目名/` 只保留项目入口、长期计划、项目工作记录和对外周报
- `weekly-reports/` 只用于对公司输出的周报
- `plans/` 只保留跨天、跨周、跨月的计划或方案

## 项目目录建议

```text
01-项目/项目名/
├── README.md
├── plans/
├── worklog/
└── weekly-reports/
```

## 使用方式

1. 每天先记录到 `05-每日/`
2. 每日中的项目工作使用 `## [[项目名]]` 分块
3. 每周将项目块归档到对应项目的 `worklog/`
4. 阶段性方案和路线设计放到 `plans/`
5. 对公司提交内容放到 `weekly-reports/`
