## 💻 开发记录

### 【water-web】地图区域鼠标悬停高亮效果

**项目**：water-web（水务 GIS 系统）
**文件**：`src/views/homePage1/index.vue`

#### 需求描述

用户希望鼠标移入地图上的区域（多边形）时，能有明显的视觉反馈，感知到鼠标已进入该区域。

#### 实现方案对比

| 方案         | 描述                     | 结论        |
| ------------ | ------------------------ | ----------- |
| 图形放大     | 以质心为基准放大多边形   | ❌ 太复杂   |
| 黑色粗边框   | 边框加粗+黑色阴影        | ❌ 不美观   |
| **浅色描边** | 基于原始颜色加亮作为边框 | ✅ 最终采用 |

#### 最终实现

**1. 颜色加亮函数**

```typescript
function lightenColor(color: any, factor: number = 0.4): number[] {
  let r: number, g: number, b: number, a: number;

  // 处理 ArcGIS Color 对象
  if (color && typeof color === "object" && "r" in color) {
    r = color.r;
    g = color.g;
    b = color.b;
    a = color.a ?? 1;
  }
  // 处理数组格式
  else if (Array.isArray(color)) {
    [r, g, b, a] = color;
    a = a ?? 1;
  } else {
    return [255, 255, 255, 0.8];
  }

  return [
    Math.min(255, Math.round(r + (255 - r) * factor)),
    Math.min(255, Math.round(g + (255 - g) * factor)),
    Math.min(255, Math.round(b + (255 - b) * factor)),
    Math.min(1, a + 0.1),
  ];
}
```

**2. 鼠标悬停交互逻辑**

- 使用 `mapView.hitTest()` 检测鼠标下方的图形
- 鼠标移入时：保存原始样式，应用浅色描边（3px）
- 鼠标移出时：恢复原始样式
- 鼠标指针自动切换（手型↔箭头）

#### 效果展示

| 状态 | 边框颜色            | 边框宽度 | 鼠标指针 |
| ---- | ------------------- | -------- | -------- |
| 默认 | 与填充同色          | 2px      | 箭头     |
| 悬停 | 浅色描边（加亮40%） | 3px      | 手型     |

#### 遇到的问题

1. **ArcGIS Color 对象格式**：`symbol.outline.color` 返回的是 Color 对象（`{r, g, b, a}`）而非数组，需要特殊处理
2. **调试过程**：添加了 console.log 发现颜色格式问题，修复后移除调试代码
