## 📝 随想记录

今天开始正式使用 Obsidian 来管理知识。PARA 方法看起来很实用：
- **Projects** - 有截止日期的项目
- **Areas** - 持续关注的领域
- **Resources** - 参考资料
- **Archive** - 归档内容

下午研究了 GitHub 上的优秀 Obsidian 案例，学到了很多：
- 不要过度使用插件，保持简单
- 让结构自然演化，不要一开始就太复杂
- Zettelkasten 方法很适合建立知识网络
- 可以用 Git 免费同步，不需要付费订阅
