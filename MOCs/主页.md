## 🔄 常用工作流

### 每日流程
1. **晨间**：处理 Inbox，计划今日
2. **日间**：捕获想法，记录笔记
3. **晚间**：写日记，复盘反思

### 知识管理流程
```
捕获 → Inbox → 处理 → 永久笔记 → 链接 → MOC
```
